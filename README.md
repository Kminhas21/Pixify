# PixelDraw - Pixel Art Editor

A lightweight, browser-based pixel art editor designed for artists, hobbyists, and game developers.

## Features

### Drawing Tools
- **Pen Tool** - Draw pixel-by-pixel or click-and-drag to paint
- **Eraser Tool** - Remove pixel colors with the same intuitive behavior
- **Color Picker (Eyedropper)** - Select colors from existing pixels
- **Fill Bucket** - Fill areas with color

### Canvas Options
- Multiple canvas sizes: 16×16, 32×32, 64×64, 128×128
- Adjustable pixel scale for comfortable viewing
- Toggle grid overlay for precision work

### Color System
- Full HSL color picker with sliders
- Hex color input for precise colors
- 20 preset colors
- Recent colors history (last 10 colors used)

### Editing Features
- Undo/Redo (up to 20 steps)
- Clear canvas
- Smooth click-and-drag drawing

### Export
- Export as PNG with transparency
- Multiple scale options: 1×, 2×, 4×, 8×, 10×

## How to Use

1. **Open the application**: Simply open `index.html` in a modern web browser
2. **Select a tool**: Click on Pen, Eraser, Eyedropper, or Fill from the left toolbar
3. **Choose a color**: Use the color sliders, presets, or hex input on the right panel
4. **Draw**: Click and drag on the canvas to create your pixel art
5. **Export**: Click "Export PNG" to save your artwork

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled

## Quick Start

```bash
# Clone the repository
git clone <repository-url>

# Open in browser
open index.html
```

Or simply drag and drop `index.html` into your browser.

## Technical Details

- Pure JavaScript (no frameworks required)
- HTML5 Canvas for rendering
- Responsive design
- High-performance pixel updates
- Efficient data structure (2D array)
